# Prepare the reference domains
1. Get protein sequences from target BGC:
./NCBI_download_refs.sh [-a accession] [-t GenBank title] [-d protein] [-o output_prefix]
Example:
./NCBI_download_refs.sh -a MH341391.1 -t "Heteroscytonema crispum UCFS10 saxitoxin biosynthetic gene cluster" -d protein -o Hcrispum_UCFS10_saxi_cluster
Optionally:
Download proteins and feature table from NCBI browser, protein database. The feature table has more details.

2. Get reference BGC InterPro domains:
a. for GenBank feature table downloaded using NCBI_download_refs.sh:
./get_genbank_reference_bgc_domains.sh <target_protein_file> <feature table> <evalue> <interpro executable path> <threads>
Example:
./get_genbank_reference_bgc_domains.sh Hcrispum_UCFS10_saxi_cluster.faa Hcrispum_UCFS10_saxi_cluster.ft 0.05 /home/marlaux/interproscan-5.76-107.0 20

b. for feature table downloaded from NCBI browser, protein database:
First convert feature table to tabular format:
./convert_genbank_features_to_table.sh Hcrispum_UCFS10_saxi_cluster_features.txt
This will generate Hcrispum_UCFS10_saxi_cluster_features.tab
./get_custom_reference_bgc_domains.sh <target_protein_file> <converted feature table> <evalue> <interpro executable path> <threads>
Example:
./get_custom_reference_bgc_domains.sh Hcrispum_UCFS10_saxi_cluster.faa Hcrispum_UCFS10_saxi_cluster_features.tab 0.05 /home/marlaux/interproscan-5.76-107.0 20
